### Quick Start

1. **Install pnpm:**

```bash
  npm install -g pnpm
```

2. **Install dependencies:**

```bash
  pnpm install
```

3. **Configure the bot:**

```bash
  cp src/config.example.ts src/config.ts
  # Edit src/config.ts with your settings
```

4. **Build the bot:**

```bash
  pnpm build
```

5. **Register Discord commands:**

```bash
  pnpm reg
```

6. **Start the bot:**

```bash
  pnpm start
```

### Development

For development with auto-reload:

```bash
  pnpm dev
```
